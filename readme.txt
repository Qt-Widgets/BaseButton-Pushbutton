BaseButton是一个建立在QLabel类上的基类按钮，实现了一些常见按钮样式的封装。

1.提供构建不同类型按钮(纯文本显示、纯图标显示、图标+文本(上下结构))的接口，并可以通过样式表和布局完成更多的细节设置。
2.为"单选按钮组"设计了一些专有属性，并完成了相关样式的处理，方便后期对此类需求的代码设计。

该基类按钮功能主要由BaseButton类实现(Widget类用来测试basebutton类的接口功能)，通过构造函数确定按钮的类型以及“单选按钮组”的专有属性。对外提供了如下几个接口：

//设置按钮三种状态（正常/鼠标按下/鼠标释放）的样式表,不设置则会使用一组代码中默认的样式,可以通过调用该接口设置按钮在不同状态下的显示细节
setBtnStyleSheet()
//选择按钮的当前样式，不同的鼠标事件会给该函数传递不同的参数，实现状态样式的切换。同时也可以在类外调用，实现非鼠标事件下的状态样式切换
selectBtnStyleSheet()
//设置按钮的显示文本
setBtnText()
//设置按钮图标显示
setBtnImage()
//获取按钮索引
getBtnIndex()
